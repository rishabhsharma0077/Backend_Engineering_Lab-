# Java & Spring Boot Mini Projects Collection

## 📖 Description
This repository contains multiple mini and major projects developed using
Java, Spring Boot, REST APIs, and related technologies.

The purpose of this repository is to demonstrate my hands-on experience
in backend and full-stack development as an entry-level Java Developer.

## ▶ How to Run Any Project
1. Clone the repository  
   git clone https://github.com/rishabhsharma0077/Backend_Engineering_Lab-

2. Open the required project folder in IDE (IntelliJ / Eclipse)

3. Configure database details in application.properties (if required)

4. Run as:
   - Java Application (Core Java projects)
   - Spring Boot Application (Spring projects)
